## 🚀 极速部署

本项目为百分之百的客户端纯原生代码架构，**无任何冗余后端依赖与 NPM 包捆绑**：

1. **本地极速预览**: 
   直接在操作系统中双击 `index.html`，或启动原生简易服务器：
   ```bash
   python3 -m http.server 8080
   ```
2. **云端原生上线**:
