# 行为守则 (CODE_OF_CONDUCT.md)

## 职责
存放各领域的编码规范与审美约束。

## 前端规范
- HTML：语义化标签，页面结构清晰。
- CSS：全局使用 Flex/Grid 布局，抛弃浮动式设计，保证良好的响应性配置。
- JS：全面拥抱 ES6 及其以上现代 JS 标准，保持模块化。

## 审美约束
- 整体风格：山海经卡通风格。
- 核心色调：
  - 森林绿（背景）：`#0a5c36`
  - 浅草绿（卡片背景）：`#e8f4e8`
  - 明亮黄（强调色）：`#f5c542`
- 圆角元素：容器级别全局适用 `border-radius: 16px`。
- 阴影体系：柔和阴影，增加拟物感。
- 字体设定：选用偏中式古风及可爱的圆体（如 ZCOOL KuaiLe 或兼容的通用网络字体）。
